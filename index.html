About

Fast API built on the Cloud

Everything for this API is hosted by Cloudflare. Using Pages, Workers and KV store, the webpage and API use as much as possible from Cloudflare's worldwide infrastructure to minimise latency.

Cloudflare's Edge nodes exist on almost every continent (sorry Antarticans!), so the latency because you and the time zone results are purely the time to calculate the result and your network.
